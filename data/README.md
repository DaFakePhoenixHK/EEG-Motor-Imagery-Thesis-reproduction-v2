# BCI Competition IV-2a data

Place the **unzipped** dataset here so this folder contains `A01T.mat`, `A01E.mat`, … `A09T.mat`, `A09E.mat` (or the usual BNCI layout with subject subfolders).

Default path expected by the benchmark:

`file_ver2/data/bci2a/`

You can also pass `--data` explicitly to `run_benchmark.py`.

**Do not commit** `.mat` files to git (they are large). This folder is gitignored except this README.
